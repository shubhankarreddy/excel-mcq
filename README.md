# Excel MCQ Skills Assessment

A practical Excel multiple-choice quiz webpage covering Basic (E1), Intermediate (E2), and Advanced (E3) topics.

Live site: https://shubhankarreddy.github.io/excel-mcq/

## Contents

- 25 questions per level
- Industry-relevant Excel scenarios
- Immediate feedback with explanations
- Progress tracking and score summary

## Levels

- Basic (E1): interface, formatting, formulas, references, filtering, printing, and workbook basics
- Intermediate (E2): lookups, charts, tables, cleaning, dates, PivotTables, validation, and conditional formatting
- Advanced (E3): XLOOKUP, dynamic arrays, Power Pivot, Power Query, protection, VBA, statistics, and dashboards
